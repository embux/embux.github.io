Yocto 1.6.2 running on ICM-2010

Posted on 2015-03-12 | In events

EMBUX is glad to introduce the yocto 1.6.2 test build running on the ICM-2010.

ICM-2010 is a 2.5’’ Pico-ITX ARM-based embedded board, powered by Freescale i.MX6, provides multi-integrated functions for easy implementation.

Please refer EMBUX website for latest update.

Information

  • uboot: 2014.04
  • kernel: 3.10.53
  • rootfs: Yocto 1.6.2 (Daisy)

Below is the introduction for Yocto 1.6.2 (Daisy) from NXP (Freescale) document:

  • What is the Yocto Project?

    “A Linux Foundation workgroup whose goal is to produce tools and processes that enable the creation of Linux distributions for embedded software independent of architecture”

  • NXP (Freescale) Yocto Project

    • Linux distributed build system
    • Replacement for LTIB
    • Higher integration with community
    • Flexible release targets

ICM-2010 Features

  • Freescale Cortex-A9 i.MX6 DualLite 1.0 GHz
  • On-board DDR3 1GB
  • 1 x 18/24 bit LVDS output
  • 1 x HDMI 1920x1080
  • 1 x 10/100/1000 Mbps Ethelrnet
  • 3 x USB 2.0 (2 external + 1 internal)
  • 1 x RS-232 (8-wire), 1 x RS-485, 1 x CAN
  • 1 x mini-PCIe for WLAN/WWAN support
  • 1 x standard SD slot
  • 1 x EMBUX X-Port for EiS (EMBUX intelligent system)
  • Multiple OS support in Linux/Android
  • Low power consumption, 5V DC input

Embedded World 2015

Posted on 2015-01-30 | In events

Welcome to join us at the Embedded World 2015 in Nuremberg, Germany, on February 24 - 26, 2015. You can find EMBUX in Hall2, Booth358.

This year at Embedded World Exhibition, Embux focuses itself on the Pico-ITX ARM-based embedded board with robust and rich I/O design. The RISC-based Single Board Computers (SBC) powered by Freescale ARM, provides multi-integrated functions for easy implementation. Embux meet the requirements for power and performance optimization, and help customers rapidly develop their own innovations in industrial control, home automation, medical device, HMI/KIOSK, digital signage, robotics and transportation segments.

Android 4.3 running on ICM-2010

Posted on 2014-12-30 | In events

We just wanted to let you know that EMBUX ICM-2010, 2.5’’ Pico-ITX ARM-based embedded board, combine with Android 4.3 is ready for evaluation.

Please refer EMBUX website for latest update.

Information

  • uboot: 2009.08
  • kernel: 3.0.35
  • rootfs: Android 4.3 AOSP
  • platform: ICM-2010

Below is the list of new features for Android 4.3 (Jelly Bean) from release notes in android website :

  • Graphics : OpenGL ES 3.0 - Android now supports the latest version of the industry standard for high performance graphics.

  • Input : Easier text input - an improved algorithm for tap-typing recognition makes text input easier.

  • Location : Location detection through Wi-Fi - use Wi-Fi to detect location without turning on Wi-Fi all the time.

  • Networking : Include NXP (Freescale) Ethernet App (ethernet Static IP/DHCP configuration). For WiFi/Bluetooth support, it will be depends on the request from customer.

  • Settings : Disabled apps tab - check which apps are disabled in Settings > Apps.

  • System : Restricted profiles - put your tablet into a mode with limited access to apps and content.

Ubuntu 12.04 running on ICM-2010

Posted on 2014-11-28 | In events

EMBUX is pleased to announce to the customers the evaluation build of Ubuntu 12.04 for ICM-2010.

ICM-2010 is a 2.5’’ Pico-ITX ARM-based embedded board, powered by Freescale i.MX6, provides multi-integrated functions for easy implementation.

Please refer EMBUX website for latest update.

Information

  • uboot: 2009.08
  • kernel: 3.0.35
  • rootfs: linaro ubuntu 12.04

Below is the list of new features for Ubuntu 12.04 (Precise Pangolin) from release notes in ubuntu website :

  • Ubuntu Software Center: The Ubuntu Software Center now offers personalized software recommendations. To use it, just turn on the Recommendations button at the bottom of the Ubuntu Software Center.

  • The HUD: The Heads-Up Display (HUD) is a easy way for user to find appropriately application by typing few letters. It provides a text-based interface comparing to the graphical menus. The applications that supports Ubuntu’s global menu bar will support the HUD.

  • Unity Settings: Ubuntu now provide the configurtion options for Unity. It is in the ystem Settings window’s Appearance panel.

  • Quicklists: Ubuntu put the frequently used options in to the pop-up menu while user right-click an application icon on the Unity. (ex: The Nautilus file manager might displays the bookmarked locations)

  • Video Lens: From Unity’s dash, the Precise Pangolin provide a quick way to search the local or online location videos. And the filter option is provide to search from a specific location.
